OnlineOrdering Lai Food
=========================
**-- A Spring and Hibernate based online food ordering system.**

Basicly, this is a simulate Doordash function but as a simplified version.  

- Presentation tier: 
    react, Spring MV
- Logic tier: 
    Spring, Spring security
- Data tier: 
    Hibernate framework
